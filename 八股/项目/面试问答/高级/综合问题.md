# 五、面试官可能重点怀疑的点

你简历写得比较丰富，面试官可能会重点验证“是不是你真的做过”。这些问题要特别准备：

1. **MSW 真的用过吗？**  
    面试官可能让你说出 Service Worker 如何注册、如何拦截请求、如何区分开发和生产环境。
2. **WebSocket 不是只会说概念。**  
    一定要准备：心跳、重连、消息补偿、重复消息、页面切换、token 过期。
3. **SKU 算法要能手写。**  
    这是电商项目最容易被要求现场推导的点。要能讲清楚路径映射和幂集。
4. **RAG 项目不能只停留在“调用大模型”。**  
    要能讲清楚：文档解析、分块、embedding、召回、topK、prompt 拼接、流式输出。
5. **性能优化不能只说“懒加载、压缩、骨架屏”。**  
    要能说出优化前后的指标、为什么有效、怎么验证。
6. **AI 导购要避免被追问穿。**  
    如果主要是规则匹配 + Ollama 流式回复，就不要包装成复杂推荐系统。可以说：“我的实现偏应用层导购，不是完整推荐算法系统。”

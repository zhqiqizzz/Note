JavaScript 数组是开发中最常用的数据结构之一，内置了丰富的方法来处理各种操作。以下是**数组方法的全面详解**，按「是否改变原数组」分类，附语法、示例、注意事项和应用场景，帮你彻底掌握数组操作。

## 一、不改变原数组的方法

|方法分类|方法名|核心作用|关键注意事项|示例用法|
|---|---|---|---|---|
|**遍历转换类**|`forEach()`|遍历数组，对每个元素执行回调|无法用`break/continue`中断，仅用于执行副作用|`javascript const nums = [1,2,3]; nums.forEach((n, i) => console.log(i, n));`|
||`map()`|遍历数组，对每个元素做映射转换|必须有`return`返回新元素，适合数据格式转换|`javascript const nums = [1,2,3]; const doubled = nums.map(n => n * 2);`|
||`filter()`|筛选数组中符合条件的元素|无符合条件的元素时返回空数组|`javascript const nums = [1,2,3,4,5]; const evens = nums.filter(n => n % 2 === 0);`|
||`reduce()`|遍历数组，将多个值聚合为单个值|建议始终传入初始值，避免空数组报错|`javascript const nums = [1,2,3,4]; const sum = nums.reduce((a, b) => a + b, 0);`|
|**查找判断类**|`find()`|查找**第一个**符合条件的元素|找到第一个匹配项后立即停止遍历|`javascript const users = [{id:1, name:'张三'}, {id:2, name:'李四'}]; const user = users.find(u => u.id === 2);`|
||`findIndex()`|查找**第一个**符合条件的元素索引|与`find()`对应，仅返回索引|`javascript const users = [{id:1, name:'张三'}, {id:2, name:'李四'}]; const index = users.findIndex(u => u.id === 2);`|
||`includes()`|判断数组是否包含指定值|可正确识别`NaN`，比`indexOf`更直观|`javascript const nums = [1,2,3,NaN]; console.log(nums.includes(2)); console.log(nums.includes(NaN));`|
||`some()`|判断数组是否**至少有一个**元素符合条件|一真即真，找到第一个匹配项立即短路|`javascript const nums = [1,2,3,4]; console.log(nums.some(n => n > 3));`|
||`every()`|判断数组是否**所有元素**都符合条件|一假即假，找到第一个不匹配项立即短路|`javascript const nums = [1,2,3,4]; console.log(nums.every(n => n > 0));`|
|**截取合并类**|`slice()`|截取数组的指定区间|支持负数索引，不传参数可实现数组浅拷贝|`javascript const nums = [1,2,3,4,5]; const sub = nums.slice(1, 4); const copy = nums.slice();`|
|**转换类**|`join()`|将数组元素按指定分隔符拼接为字符串|默认分隔符为逗号`,`，传空字符串可直接拼接|`javascript const arr = ['a','b','c']; const str1 = arr.join(); const str2 = arr.join('-');`|
|**扁平化类**|`flat()`|将多维数组扁平化为一维数组|默认扁平化深度为 1，传`Infinity`可扁平化任意深度|`javascript const arr = [1, [2, [3, 4]], 5]; const flat1 = arr.flat(); const flat2 = arr.flat(2);`|
||`flatMap()`|先执行`map()`映射，再执行`flat()`扁平化|比分别调用`map+flat`性能更高，深度固定为 1|`javascript const strs = ['hello world', 'foo bar']; const words = strs.flatMap(s => s.split(' '));`|
|**ES2023 新增**|`toReversed()`|反转数组顺序|替代`reverse()`，不修改原数组|`javascript const nums = [1,2,3]; const reversed = nums.toReversed();`|
||`toSorted()`|对数组进行排序|替代`sort()`，不修改原数组，支持自定义比较函数|`javascript const nums = [3,1,4,1,5]; const sorted = nums.toSorted((a,b) => a - b);`|

## 二、改变原数组的方法

| 方法分类         | 方法名         | 核心作用                 | 关键注意事项                               | 示例用法                                                                                                                            |
| ------------ | ----------- | -------------------- | ------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------- |
| **栈 / 队列操作** | `push()`    | 向数组**末尾**添加一个 / 多个元素 | 性能优于`unshift()`，不会改变其他元素索引           | `javascript const nums = [1,2,3]; nums.push(4, 5);`                                                                             |
|              | `pop()`     | 删除数组**末尾最后一个**元素     | 空数组调用返回`undefined`                   | `javascript const nums = [1,2,3]; const last = nums.pop();`                                                                     |
|              | `unshift()` | 向数组**开头**添加一个 / 多个元素 | 会改变所有元素的索引，大数组性能较差                   | `javascript const nums = [1,2,3]; nums.unshift(-1, 0);`                                                                         |
|              | `shift()`   | 删除数组**开头第一个**元素      | 会改变所有元素的索引，大数组性能较差                   | `javascript const nums = [1,2,3]; const first = nums.shift();`                                                                  |
| **万能增删改**    | `splice()`  | 对数组进行删除、插入、替换操作      | 唯一可同时实现增删改的数组方法                      | `javascript const nums = [1,2,3,4]; // 删除 nums.splice(1, 2); // 插入 nums.splice(1, 0, 2, 3); // 替换 nums.splice(1, 2, '二', '三');` |
| **排序与反转**    | `sort()`    | 对数组进行排序              | 默认按字符串 Unicode 码点排序，**数字排序必须传入比较函数** | `javascript const nums = [3,1,4,1,5]; // 升序 nums.sort((a, b) => a - b); // 降序 nums.sort((a, b) => b - a);`                      |
|              | `reverse()` | 反转数组的元素顺序            | 推荐用 ES2023 的`toReversed()`替代         | `javascript const nums = [1,2,3]; nums.reverse();`                                                                              |
| **填充**       | `fill()`    | 用指定值填充数组的指定区间        | 常用于初始化固定长度的数组                        | `javascript const arr = new Array(5).fill(0); const nums = [1,2,3,4,5]; nums.fill(0, 1, 4);`                                    |
## 三、数组静态方法（直接通过`Array.xxx`调用）

|方法名|核心作用|关键注意事项| 示例用法                                                                                                                                                                   |
|---|---|---|---|
|`Array.isArray()`|判断一个值是否为数组|最可靠的数组类型判断方法| `javascript console.log(Array.isArray([1,2,3])); console.log(Array.isArray({}));`                                                                                      |
|`Array.from()`|将类数组对象 / 可迭代对象转为真正的数组|支持第二个参数（类似`map`的回调），可用于去重| `javascript // 字符串转数组 const arr1 = Array.from('hello'); // Set去重 const arr2 = Array.from(new Set([1,2,2,3])); // 带map回调 const arr3 = Array.from([1,2,3], n => n * 2);` |
|`Array.of()`|根据传入的参数创建数组|解决`Array(3)`创建空数组的怪异行为| `javascript console.log(Array.of(3)); console.log(Array.of(1,2,3));                                                                                                    
## 五、易混淆方法对比

### 1. `map()` vs `forEach()`

|特性|`map()`|`forEach()`|
|---|---|---|
|返回值|新数组|`undefined`|
|用途|生成新数组|执行副作用|
|推荐场景|数据转换、提取属性|打印、修改外部变量|
### 2. `filter()` vs `find()`

|特性|`filter()`|`find()`|
|---|---|---|
|返回值|满足条件的元素组成的数组|第一个满足条件的元素|
|结果数量|可能多个|最多一个|
|找不到时|空数组|`undefined`|
### 3. `slice()` vs `splice()`

|特性|`slice()`|`splice()`|
|---|---|---|
|是否改变原数组|否|是|
|作用|截取数组|增删改数组元素|
|返回值|截取的新数组|被删除元素组成的数组|

## 六、最佳实践总结

1. **优先使用不改变原数组的方法**：如 `map()`、`filter()`、`slice()`、`toReversed()` 等，更安全，避免副作用。
2. **根据需求选择合适的方法**：
    
    - 数据转换 → `map()`
    - 筛选数据 → `filter()`
    - 聚合数据 → `reduce()`
    - 查找元素 → `find()`/`findIndex()`
    - 判断条件 → `some()`/`every()`
    - 数组去重 → `[...new Set(arr)]` 或 `Array.from(new Set(arr))`
    
3. **注意性能**：
    
    - 大数组避免用 `unshift()`/`shift()`（会改变所有元素索引）。
    - 复杂聚合用 `reduce()`，简单循环用 `for`（性能更好）。
    
4. **ES2023 新方法**：优先用 `toReversed()`、`toSorted()`、`toSpliced()`、`with()` 替代原来的 `reverse()`、`sort()`、`splice()`，避免改变原数组。

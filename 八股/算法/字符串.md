以下是字符串方法的完整表格整理，按功能分类，核心信息一目了然，方便快速查阅和上手。

## 一、截取与分割类

| 方法名                       | 核心作用            | 关键注意事项                               | 示例用法                                                                                              |
| ------------------------- | --------------- | ------------------------------------ | ------------------------------------------------------------------------------------------------- |
| **split()**               | 按指定分隔符将字符串分割为数组 | 支持正则表达式作为分隔符，传空字符串可将每个字符拆分为数组元素      | `javascript const str = 'a,b,c,d'; const arr1 = str.split(','); const arr2 = 'hello'.split('');`  |
| **slice(start, end)**     | 截取字符串的指定区间      | 支持负数索引（从末尾计数），`end` 可选（默认到末尾），推荐优先使用 | `javascript const str = 'hello world'; console.log(str.slice(0, 5)); console.log(str.slice(-5));` |
| **substring(start, end)** | 截取字符串的指定区间      | 不支持负数索引，若 `start > end` 会自动交换两者位置    | `javascript const str = 'hello world'; console.log(str.substring(0, 5));`                         |

## 二、查找与判断类

| 方法名                           | 核心作用                   | 关键注意事项                                  | 示例用法                                                                                                          |
| ----------------------------- | ---------------------- | --------------------------------------- | ------------------------------------------------------------------------------------------------------------- |
| **indexOf(str, fromIndex)**   | 查找子串**第一次出现**的索引       | 找不到返回 `-1`，可通过 `fromIndex` 指定查找起始位置     | `javascript const str = 'hello hello'; console.log(str.indexOf('hello')); console.log(str.indexOf('world'));` |
| `lastIndexOf(str, fromIndex)` | 查找子串**最后一次出现**的索引      | 从数组末尾向前查找，找不到返回 `-1`                    | `javascript const str = 'hello hello'; console.log(str.lastIndexOf('hello'));`                                |
| `includes(str, fromIndex)`    | 判断字符串是否包含指定子串          | ES2016 新增，返回布尔值，比 `indexOf` 更直观         | `javascript const str = 'hello world'; console.log(str.includes('world')); console.log(str.includes('vue'));` |
| `startsWith(str, fromIndex)`  | 判断字符串是否以指定子串**开头**     | ES2015 新增，返回布尔值                         | `javascript const str = 'https://vuejs.org'; console.log(str.startsWith('https://'));`                        |
| `endsWith(str, length)`       | 判断字符串是否以指定子串**结尾**     | ES2015 新增，返回布尔值，可通过 `length` 指定判断的字符串长度 | `javascript const str = 'https://vuejs.org'; console.log(str.endsWith('.org'));`                              |
| `charAt(index)`               | 获取指定索引位置的字符            | 索引超出范围返回空字符串                            | `javascript const str = 'hello'; console.log(str.charAt(0));`                                                 |
| `charCodeAt(index)`           | 获取指定索引位置字符的 Unicode 编码 | 索引超出范围返回 `NaN`                          | `javascript const str = 'hello'; console.log(str.charCodeAt(0));`                                             |
## 三、替换与修改类

|方法名|核心作用|关键注意事项|示例用法|
|---|---|---|---|
|`replace(search, replace)`|替换**第一个**匹配的子串|支持正则表达式，若需替换所有匹配需用 `/g` 修饰符|`javascript const str = 'hello hello'; console.log(str.replace('hello', 'hi')); console.log(str.replace(/hello/g, 'hi'));`|
|`replaceAll(search, replace)`|替换**所有**匹配的子串|ES2021 新增，无需正则 `/g` 即可替换所有|`javascript const str = 'hello hello'; console.log(str.replaceAll('hello', 'hi'));`|
|`trim()`|去除字符串**首尾**的空白字符（空格、换行、制表符等）|不会去除字符串中间的空白|`javascript const str = ' hello world '; console.log(str.trim());`|
|`trimStart()` / `trimLeft()`|仅去除字符串**开头**的空白字符|ES2019 新增，`trimLeft()` 是别名|`javascript const str = ' hello world '; console.log(str.trimStart());`|
|`trimEnd()` / `trimRight()`|仅去除字符串**结尾**的空白字符|ES2019 新增，`trimRight()` 是别名|`javascript const str = ' hello world '; console.log(str.trimEnd());`|
|`toUpperCase()`|将字符串转换为**大写**|不会改变原字符串，返回新字符串|`javascript const str = 'Hello World'; console.log(str.toUpperCase());`|
|`toLowerCase()`|将字符串转换为**小写**|不会改变原字符串，返回新字符串|`javascript const str = 'Hello World'; console.log(str.toLowerCase());`|
|`padStart(targetLength, padString)`|在字符串**开头**补全指定字符，使字符串达到目标长度|ES2017 新增，`padString` 可选，默认补空格|`javascript const str = '5'; console.log(str.padStart(2, '0'));`|
|`padEnd(targetLength, padString)`|在字符串**结尾**补全指定字符，使字符串达到目标长度|ES2017 新增，`padString` 可选，默认补空格|`javascript const str = 'hello'; console.log(str.padEnd(10, '*'));`|
## 四、拼接与重复类

|方法名|核心作用|关键注意事项|示例用法|
|---|---|---|---|
|`concat(str1, str2, ...)`|拼接两个或多个字符串|不会改变原字符串，返回新字符串，更推荐用 `+` 或模板字符串|`javascript const str1 = 'hello'; const str2 = 'world'; console.log(str1.concat(' ', str2));`|
|`repeat(count)`|将字符串重复指定次数|ES2015 新增，`count` 必须是非负整数，否则报错|`javascript const str = 'ha'; console.log(str.repeat(3));`|
## 五、其他常用方法

|方法名|核心作用|关键注意事项|示例用法|
|---|---|---|---|
|`toString()`|返回字符串本身|通常用于将其他类型（如数字、数组）转为字符串|`javascript const num = 123; console.log(num.toString());`|
|`valueOf()`|返回字符串的原始值|通常由 JS 引擎自动调用，无需手动使用|`javascript const str = 'hello'; console.log(str.valueOf());`|
|`localeCompare(compareString)`|比较两个字符串的排序顺序（考虑本地化）|返回负数、0 或正数，分别表示小于、等于、大于|`javascript const str1 = 'apple'; const str2 = 'banana'; console.log(str1.localeCompare(str2));`|

在 JavaScript 中，函数是特殊的对象，因此拥有一些实用的**实例方法**（通过函数实例调用）。其中最核心、最常用的是**改变 `this` 指向**的三个方法：`call()`、`apply()` 和 `bind()`。以下是详细介绍：

### 一、改变 `this` 指向的核心方法

这三个方法的作用都是**显式指定函数执行时的 `this` 值**，但在传参方式和执行时机上有明显区别。

#### 1. `call(thisArg, arg1, arg2, ...)`

- **作用**：**立即执行函数**，指定函数内部的 `this` 为 `thisArg`，并**逐个传入参数**。
    
- **参数**：
    
    - `thisArg`：函数执行时的 `this` 值（非严格模式下 `null`/`undefined` 会指向全局对象）。
    - `arg1, arg2, ...`：函数执行所需的参数，逐个传入。
    
- **示例**：

```javascript
    function greet(greeting, punctuation) {
      console.log(`${greeting}, ${this.name}${punctuation}`);
    }
    
    const person = { name: 'Alice' };
    // 立即执行 greet，this 指向 person，参数逐个传入
    greet.call(person, 'Hello', '!'); // 输出：Hello, Alice!
```
    
- **经典场景**：借用其他对象的方法（如类数组借用数组方法）。
    
```javascript
    const arrayLike = { 0: 'a', 1: 'b', length: 2 };
    // 借用数组的 push 方法
    Array.prototype.push.call(arrayLike, 'c');
    console.log(arrayLike); // { 0: 'a', 1: 'b', 2: 'c', length: 3 }
```

#### 2. `apply(thisArg, [argsArray])`

- **作用**：**立即执行函数**，指定函数内部的 `this` 为 `thisArg`，并**以数组 / 类数组形式传入参数**。
    
- **参数**：
    
    - `thisArg`：函数执行时的 `this` 值。
    - `[argsArray]`：函数执行所需的参数，以数组或类数组形式传入。
    
- **示例**：
    
```javascript
    function sum(a, b, c) {
      return a + b + c;
    }
    
    // 立即执行 sum，this 指向 null，参数以数组传入
    const result = sum.apply(null, [1, 2, 3]);
    console.log(result); // 6
```
    
- **经典场景**：处理数组相关的操作（如求数组最大值）。
    
```javascript
    const numbers = [5, 2, 9, 1];
    // 借用 Math.max，将数组作为参数传入
    const max = Math.max.apply(null, numbers);
    console.log(max); // 9
```

#### 3. `bind(thisArg, arg1, arg2, ...)`

- **作用**：**创建一个新函数**（不立即执行），指定新函数内部的 `this` 为 `thisArg`，并可**预设部分参数**（偏函数）。
    
- **参数**：
    
    - `thisArg`：新函数执行时的 `this` 值。
    - `arg1, arg2, ...`：预设的参数，新函数执行时会先使用这些参数。
    
- **返回值**：一个新的绑定函数。
    
- **示例**：
    
```javascript
    function greet(greeting, punctuation) {
      console.log(`${greeting}, ${this.name}${punctuation}`);
    }
    
    const person = { name: 'Bob' };
    // 创建新函数 boundGreet，this 绑定到 person，并预设第一个参数为 'Hi'
    const boundGreet = greet.bind(person, 'Hi');
    // 执行新函数，只需传入剩余参数
    boundGreet('~'); // 输出：Hi, Bob~
```
    
- **经典场景**：绑定回调函数的 `this`（如定时器、事件处理）。
    
```javascript
    const counter = {
      count: 0,
      start() {
        // 绑定 this 到 counter，避免定时器中 this 指向全局对象
        setInterval(function() {
          this.count++;
          console.log(this.count);
        }.bind(this), 1000);
      }
    };
    counter.start(); // 每秒输出 1, 2, 3...
```

### 二、`call()`、`apply()`、`bind()` 对比

|方法|执行时机|传参方式|返回值|核心场景|
|---|---|---|---|---|
|`call()`|立即执行|逐个传参|函数执行结果|借用方法、指定 `this` 传参|
|`apply()`|立即执行|数组 / 类数组传参|函数执行结果|数组操作（如求最大值、合并）|
|`bind()`|不立即执行|逐个预设参数|新的绑定函数|绑定 `this`、偏函数、回调|

### 三、其他常用方法

#### `toString()`

- **作用**：返回函数的**源代码字符串**（包含函数定义、参数、函数体）。
- **示例**：
    
```javascript
    function add(a, b) {
      return a + b;
    }
    console.log(add.toString());
    // 输出：
    // function add(a, b) {
    //   return a + b;
    // }
```
### 四、函数的常用属性（补充）

虽然不是方法，但以下属性非常实用：

- **`length`**：返回函数的**形参个数**（不包括剩余参数）。
    
```javascript
    function func(a, b, ...rest) {}
    console.log(func.length); // 2
```

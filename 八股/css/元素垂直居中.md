垂直居中是 CSS 布局中最常见的需求之一，根据元素类型（单行文本、块级元素）、高度是否固定、浏览器兼容性要求等，有多种实现方案。以下是**最常用且实用的方法**，从现代首选到传统兼容逐一介绍。

## 一、单行文本垂直居中（最简单）

利用 `line-height` 等于父元素 `height`，使文本在行高内自动垂直居中。

```css
.parent {
  height: 60px;       /* 父元素固定高度 */
  line-height: 60px;  /* 行高 = 父元素高度 */
}
```

### 适用场景

- 按钮文本、导航栏文字、标签文字等**单行内容**；
- 仅适用于文本或行内元素（如 `<span>`、`<a>`）。

## 二、Flex 布局（现代首选，推荐）

将父元素设为 **Flex 容器**，通过 `align-items` 控制交叉轴（垂直方向）对齐，是目前最通用的方案。

#### 1. 仅垂直居中

```css
.parent {
  display: flex;
  align-items: center; /* 垂直居中（交叉轴对齐） */
}
```

#### 2. 水平 + 垂直同时居中

```css
.parent {
  display: flex;
  align-items: center;     /* 垂直居中 */
  justify-content: center; /* 水平居中（主轴对齐） */
}
```

### 适用场景

- **绝大多数场景**，子元素高度**固定或不固定**均可；
- 现代浏览器（Chrome、Firefox、Safari、Edge）完美支持，IE11 部分兼容（需注意小细节）。

## 三、Grid 布局（代码最简洁）

将父元素设为 **Grid 容器**，通过 `place-items` 一次性控制水平和垂直居中，代码比 Flex 更短。

```css
.parent {
  display: grid;
  place-items: center; /* 等同于 align-items: center; justify-items: center; */
}
```

### 适用场景

- 现代浏览器（比 Flex 稍新，但主流浏览器均支持）；
- 快速实现水平垂直居中，代码极简。

## 四、绝对定位 + transform（子元素高度不固定）

子元素绝对定位后，`top: 50%` 先移到父元素垂直中线，再用 `transform: translateY(-50%)` 向上偏移**自身高度的一半**，实现居中（transform 相对于自身尺寸计算）。

```css
.parent {
  position: relative; /* 父元素设为相对定位，作为子元素的定位基准 */
}
.child {
  position: absolute;
  top: 50%;
  transform: translateY(-50%); /* 向上偏移自身高度的50% */
}
```

### 适用场景

- 子元素**高度不固定**（动态内容）；
- 兼容性较好（IE9+ 支持 transform）。

## 五、绝对定位 + margin: auto（子元素高度固定）

子元素绝对定位后，将 `top`/`bottom`/`left`/`right` 都设为 `0`，再通过 `margin: auto` 自动分配剩余空间，实现垂直 + 水平居中。

```css
.parent {
  position: relative;
}
.child {
  position: absolute;
  top: 0;
  bottom: 0;
  left: 0;
  right: 0;
  height: 100px; /* 必须有固定高度（宽度也建议固定） */
  margin: auto;
}
```

### 适用场景

- 子元素**高度固定**；
- 兼容性好（IE8+ 支持）。

## 六、table-cell 布局（兼容老浏览器）

模拟表格单元格的 `vertical-align: middle` 特性，将父元素设为 `display: table-cell`，利用表格的垂直对齐能力。

```css
/* 外层容器（可选，若需要父元素占满宽度） */
.table-wrapper {
  display: table;
  width: 100%;
}

.parent {
  display: table-cell;
  vertical-align: middle; /* 垂直居中 */
  height: 200px; /* 父元素需有固定高度 */
}
```

### 适用场景

- 需要兼容**IE8+ 等老浏览器**；
- 现代项目已较少使用，优先选 Flex/Grid。

## 七、行内块 + 伪元素（图片 / 行内块居中）

用伪元素 `::before` 撑起父元素高度，子元素设为 `display: inline-block`，通过 `vertical-align: middle` 与伪元素对齐。

```css
.parent {
  height: 200px;
  text-align: center; /* 顺便水平居中 */
}

/* 伪元素撑起高度 */
.parent::before {
  content: '';
  display: inline-block;
  height: 100%;
  vertical-align: middle;
}

.child {
  display: inline-block;
  vertical-align: middle; /* 与伪元素垂直对齐 */
}
```

### 适用场景

- 图片、图标等**行内块元素**居中；
- 兼容性好（IE8+ 支持）。

## 八、方法总结与推荐

| 方法               | 子元素高度要求  | 浏览器兼容性 | 推荐指数  | 适用场景          |
| ---------------- | -------- | ------ | ----- | ------------- |
| Flex 布局          | 固定 / 不固定 | 现代浏览器  | ⭐⭐⭐⭐⭐ | 绝大多数场景，首选方案   |
| Grid 布局          | 固定 / 不固定 | 现代浏览器  | ⭐⭐⭐⭐⭐ | 快速实现，代码极简     |
| 绝对定位 + transform | 不固定      | IE9+   | ⭐⭐⭐⭐  | 动态内容，兼容较好     |
| 绝对定位 + margin    | 固定       | IE8+   | ⭐⭐⭐   | 固定尺寸元素，老浏览器兼容 |
| table-cell       | 无要求      | IE8+   | ⭐⭐    | 老浏览器兼容，现代项目少用 |
| 行内块 + 伪元素        | 无要求      | IE8+   | ⭐⭐⭐   | 图片、行内块元素居中    |

### 核心建议

- **现代项目**：优先用 **Flex** 或 **Grid**，简单高效；
- **动态内容（高度不固定）**：Flex、Grid、绝对定位 + transform；
- **老浏览器兼容**：table-cell、绝对定位 + margin；
- **单行文本**：直接用 `line-height`。

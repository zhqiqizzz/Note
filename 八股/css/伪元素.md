是 CSS 中**创建文档树中不存在的虚拟元素**的语法，专门用于给元素的特定部分添加样式或插入装饰性内容，不改变 HTML 结构，是前端样式美化的核心利器。

## 核心概念与语法

### 定义

伪元素不是真实的 DOM 节点，而是 CSS 渲染层生成的 "虚拟节点"，只能通过 CSS 访问和修改，无法通过 JavaScript 直接操作。

### 语法规范

- **CSS3 标准**：使用**双冒号 `::`** 区分伪元素和伪类（如 `::before`）
- **兼容写法**：单冒号 `:` 也能生效（为了兼容 IE8 及以下），但强烈推荐使用双冒号

### 与伪类的本质区别（最容易混淆）

|维度|伪元素|伪类|
|---|---|---|
|**作用**|创建**不存在的虚拟元素**，操作元素的 "部分内容"|选择**已存在的真实元素**，匹配元素的 "状态"|
|**语法**|双冒号 `::`（标准）|单冒号 `:`|
|**数量**|一个元素最多同时使用 2 个（`::before` 和 `::after`）|一个元素可以同时使用多个|
|**示例**|`::before`、`::first-letter`|`:hover`、`:active`、`:nth-child()`|

## 最常用的 7 个伪元素

### 1. `::before` & `::after`（90% 场景都会用到）

在元素的**内容之前 / 之后**插入虚拟内容，是功能最强大的伪元素。

- ✅ **必须属性**：`content`（即使是空内容也要写 `content: ""`，否则伪元素不显示）
- ✅ 支持所有 CSS 属性：定位、宽高、背景、动画、渐变等
- ✅ 默认是行内元素，可通过 `display` 修改

```css
/* 示例：给按钮添加左侧图标 */
.btn::before {
  content: "🔍"; /* 可以是文本、emoji、图片url()、空字符串 */
  margin-right: 8px;
}

/* 示例：制作分割线 */
.title::after {
  content: "";
  display: block;
  width: 50px;
  height: 2px;
  background-color: #1890ff;
  margin-top: 8px;
}
```

### 2. `::first-letter`

选中元素的**第一个字符**，常用于文章首字下沉、首字母大写效果。

```css
p::first-letter {
  font-size: 3em;
  float: left;
  margin-right: 8px;
  color: #1890ff;
  font-weight: bold;
}
```

### 3. `::first-line`

选中元素的**第一行文本**，样式会随窗口宽度自动调整。

```css
p::first-line {
  font-weight: bold;
  color: #333;
}
```

### 4. `::selection`

选中用户**鼠标拖动选中的文本**，只能修改以下属性：

- `color`、`background-color`
- `text-shadow`、`cursor`、`outline`

```css
::selection {
  background-color: #1890ff;
  color: #fff;
}

/* 仅对某个元素生效 */
.article::selection {
  background-color: #f5f7fa;
  color: #333;
}
```

### 5. `::placeholder`

修改输入框 / 文本域的**占位符文字样式**，注意浏览器前缀。

```css
input::placeholder {
  color: #c0c4cc;
  font-size: 14px;
}

/* 兼容旧版浏览器 */
input::-webkit-input-placeholder { /* Chrome/Safari */
  color: #c0c4cc;
}
input::-moz-placeholder { /* Firefox */
  color: #c0c4cc;
}
```

### 6. `::marker`（CSS3 新增）

修改列表项（`<li>`）的**标记符号**（圆点、数字、自定义符号）的样式。

```css
li::marker {
  color: #1890ff;
  font-size: 1.2em;
  content: "✅ "; /* 自定义列表符号 */
}
```

### 7. `::backdrop`（CSS3 新增）

修改全屏元素（如 `<dialog>`、`video:fullscreen`）的**背景遮罩层**样式。

```css
dialog::backdrop {
  background-color: rgba(0, 0, 0, 0.7);
}
```

## 核心特性与限制

1. **继承性**：伪元素继承其父元素的可继承样式（如字体、颜色）
2. **定位**：伪元素相对于其父元素定位（父元素设为 `position: relative`）
3. **事件**：伪元素**不能绑定任何 DOM 事件**（点击、hover 等事件会透传给父元素）
4. **可访问性**：屏幕阅读器通常不会读取伪元素的 `content` 内容，**不要放重要信息**
5. **数量限制**：一个元素只能有一个 `::before` 和一个 `::after`

## 经典实战场景

### 1. 清除浮动（经典 clearfix）

```css
.clearfix::after {
  content: "";
  display: block;
  clear: both;
}
```

### 2. 制作纯 CSS 三角形

```css
.triangle {
  width: 0;
  height: 0;
  border: 10px solid transparent;
  border-bottom-color: #1890ff;
}
```

### 3. 自定义复选框

```css
.checkbox {
  position: relative;
  padding-left: 24px;
}

.checkbox::before {
  content: "";
  position: absolute;
  left: 0;
  top: 2px;
  width: 16px;
  height: 16px;
  border: 1px solid #dcdfe6;
  border-radius: 2px;
}

.checkbox:checked::after {
  content: "✓";
  position: absolute;
  left: 3px;
  top: 0;
  color: #1890ff;
  font-size: 14px;
}
```

## 最佳实践

1. **优先使用双冒号语法**，符合 CSS3 标准，便于区分伪类和伪元素
2. **伪元素只用于装饰性内容**，不要放对页面语义和功能至关重要的信息
3. **避免滥用**：复杂布局优先使用真实 DOM，伪元素过多会增加渲染负担
4. **注意兼容性**：`::marker`、`::backdrop` 等新伪元素在 IE 中不支持
5. **保持作用域清晰**：在 Vue/React 中使用时，注意样式作用域问题

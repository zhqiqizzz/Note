拖拽组件是前端开发中非常高频的需求，实现方式主要分为两类：

1. **原生手写**：理解拖拽核心原理，适合简单场景；
2. **成熟库实现**：推荐 `VueUse` 的 `useDraggable`（简单拖拽）或 `vuedraggable`（列表拖拽排序），生产环境首选。

下面结合 Vue 3 `<script setup>`，从原理到落地，给你一套完整方案。

## 一、先搞懂：拖拽的核心原理（3 个事件）

无论用什么方案，拖拽的底层逻辑都是基于鼠标事件：

1. **`mousedown`**：鼠标按下时，记录初始位置，标记 “开始拖拽”；
2. **`mousemove`**：鼠标移动时，计算偏移量，更新元素位置；
3. **`mouseup`**：鼠标松开时，移除事件监听，标记 “结束拖拽”。

## 二、方案一：原生手写（理解原理用）

适合简单的绝对定位元素拖拽，代码量不大，能帮你彻底理解拖拽逻辑。

### 完整可运行代码

```vue
<template>
  <div class="container">
    <!-- 可拖拽的方块 -->
    <div
      class="draggable-box"
      :style="{ left: x + 'px', top: y + 'px' }"
      @mousedown="onMouseDown"
    >
      拖拽我
    </div>
  </div>
</template>

<script setup>
import { ref, onUnmounted } from 'vue'

// 元素位置
const x = ref(100)
const y = ref(100)

// 拖拽状态
let isDragging = false
// 鼠标按下时的初始偏移量
let startX = 0
let startY = 0

// 1. 鼠标按下：开始拖拽
const onMouseDown = (e) => {
  isDragging = true
  // 记录初始位置：鼠标当前位置 - 元素当前位置
  startX = e.clientX - x.value
  startY = e.clientY - y.value

  // 绑定移动和松开事件（绑定到 document 上，防止鼠标移出元素后失效）
  document.addEventListener('mousemove', onMouseMove)
  document.addEventListener('mouseup', onMouseUp)
  
  // 防止拖拽时选中文本
  e.preventDefault()
}

// 2. 鼠标移动：更新位置
const onMouseMove = (e) => {
  if (!isDragging) return
  // 计算新位置
  x.value = e.clientX - startX
  y.value = e.clientY - startY
}

// 3. 鼠标松开：结束拖拽
const onMouseUp = () => {
  isDragging = false
  // 移除事件监听，防止内存泄漏
  document.removeEventListener('mousemove', onMouseMove)
  document.removeEventListener('mouseup', onMouseUp)
}

// 组件卸载时清理事件
onUnmounted(() => {
  document.removeEventListener('mousemove', onMouseMove)
  document.removeEventListener('mouseup', onMouseUp)
})
</script>

<style scoped>
.container {
  width: 100vw;
  height: 100vh;
  position: relative;
  background: #f5f5f5;
}

.draggable-box {
  width: 100px;
  height: 100px;
  background: #409eff;
  color: #fff;
  display: flex;
  align-items: center;
  justify-content: center;
  border-radius: 8px;
  cursor: move;
  position: absolute;
  user-select: none; /* 禁止选中文本 */
}
</style>
```

## 三、方案二：VueUse `useDraggable`

如果你不想手写原生逻辑，**VueUse** 是你的最佳选择，它是 Vue 3 生态中最常用的工具库，提供了 `useDraggable` 这个组合式 API，一行代码实现拖拽。

### 1. 安装 VueUse

```bash
npm i @vueuse/core
# 或 yarn
yarn add @vueuse/core
```

### 2. 完整可运行代码

```vue
<template>
  <div class="container">
    <!-- 可拖拽的方块 -->
    <div
      ref="targetRef"
      class="draggable-box"
      :style="{ left: x + 'px', top: y + 'px' }"
    >
      拖拽我 (VueUse)
    </div>
  </div>
</template>

<script setup>
import { ref } from 'vue'
import { useDraggable } from '@vueuse/core'

// 绑定拖拽元素的 ref
const targetRef = ref(null)

// 一行代码实现拖拽，返回 x, y 位置
const { x, y } = useDraggable(targetRef, {
  initialValue: { x: 100, y: 100 }, // 初始位置
  // 可选：拖拽边界限制
  // containment: { left: 0, right: 800, top: 0, bottom: 600 }
})
</script>

<style scoped>
.container {
  width: 100vw;
  height: 100vh;
  position: relative;
  background: #f5f5f5;
}

.draggable-box {
  width: 100px;
  height: 100px;
  background: #67c23a;
  color: #fff;
  display: flex;
  align-items: center;
  justify-content: center;
  border-radius: 8px;
  cursor: move;
  position: absolute;
  user-select: none;
}
</style>
```

### 3. 核心优势

- ✅ 代码极简，一行 API 搞定；
- ✅ 原生支持边界限制、拖拽手柄、事件回调；
- ✅ 自动处理事件监听和清理，无内存泄漏；
- ✅ 支持触摸事件（移动端也能用）。

## 四、方案三：`vuedraggable`（列表拖拽排序专用）

如果你的需求是**列表拖拽排序**（比如拖拽调整简历顺序、调整任务列表顺序），`vuedraggable` 是最佳选择，它基于 `Sortable.js`，专门解决列表拖拽问题。

### 1. 安装

```bash
npm i vuedraggable@next
# 或 yarn
yarn add vuedraggable@next
```

### 2. 完整可运行代码（列表拖拽排序）

```vue
<template>
  <div class="container">
    <h2>拖拽排序简历列表</h2>
    
    <!-- vuedraggable 组件，直接绑定你的数组 -->
    <draggable
      v-model="list"
      item-key="id"
      class="list"
      ghost-class="ghost"
      @start="onDragStart"
      @end="onDragEnd"
    >
      <template #item="{ element }">
        <div class="list-item">
          <span class="handle">⋮⋮</span>
          <span>{{ element.name }}</span>
        </div>
      </template>
    </draggable>

    <p class="result">当前列表顺序：{{ list.map(i => i.name).join(' → ') }}</p>
  </div>
</template>

<script setup>
import { ref } from 'vue'
import draggable from 'vuedraggable'

// 要拖拽排序的列表
const list = ref([
  { id: 1, name: '张三 - 前端工程师' },
  { id: 2, name: '李四 - 后端工程师' },
  { id: 3, name: '王五 - 产品经理' },
  { id: 4, name: '赵六 - UI设计师' }
])

// 拖拽开始回调
const onDragStart = (e) => {
  console.log('开始拖拽', e.item)
}

// 拖拽结束回调
const onDragEnd = (e) => {
  console.log('拖拽结束，新顺序：', list.value)
}
</script>

<style scoped>
.container {
  width: 500px;
  margin: 40px auto;
}
.list {
  border: 1px solid #eee;
  border-radius: 8px;
  background: #fff;
}
.list-item {
  display: flex;
  align-items: center;
  gap: 12px;
  padding: 12px 16px;
  border-bottom: 1px solid #f0f0f0;
  cursor: move;
}
.list-item:last-child {
  border-bottom: none;
}
.handle {
  color: #999;
  cursor: grab;
}
/* 拖拽时的占位符样式 */
.ghost {
  opacity: 0.5;
  background: #c8ebfb;
}
.result {
  margin-top: 20px;
  color: #666;
}
</style>
```

## 五、方案对比与选择建议

|方案|优点|缺点|适用场景|
|---|---|---|---|
|**原生手写**|理解原理，无依赖|代码繁琐，需处理边界情况（文本选中、内存泄漏、移动端）|学习原理、极简单场景|
|**VueUse `useDraggable`**|代码极简，功能强大，支持边界 / 触摸|仅支持单个元素拖拽|单个元素自由拖拽（弹窗、卡片、悬浮按钮）|
|**vuedraggable**|专门解决列表排序，功能最全|依赖 Sortable.js，包稍大|列表拖拽排序（任务列表、简历排序、图片排序）|

## 六、结合你的招聘软件场景

如果你的招聘软件需要：

1. **拖拽调整简历筛选条件的顺序**：用 `vuedraggable`；
2. **拖拽移动聊天窗口 / 悬浮按钮**：用 VueUse `useDraggable`；
3. **拖拽上传简历**：配合 `drop` 事件，或者用成熟的上传组件库。

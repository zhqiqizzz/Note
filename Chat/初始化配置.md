# 第 1 步：创建总项目目录

找一个你放代码的地方，比如桌面或开发目录，然后执行：

```
mkdir chat-appcd chat-app
```

你的项目最终会长这样：

```
chat-app/
  frontend/
  backend/
  docker-compose.yml
```

# 第 2 步：创建 Vue + TS 前端

执行：

```
npm create vue@latest frontend
```

Vue 官方快速开始也是用 `npm create vue@latest` 创建项目。

它会问你一些问题，你这样选：

```
Project name: frontend
Add TypeScript? Yes
Add JSX Support? No
Add Vue Router? Yes
Add Pinia? Yes
Add Vitest? No
Add an End-to-End Testing Solution? No
Add ESLint? Yes
Add Prettier? Yes
Add Vue DevTools? Yes 或 No 都行
```

然后进入前端项目：

```
cd frontend
npm install
npm run dev
```

看到类似这个就说明前端成功：

```
Local: http://localhost:5173/
```

浏览器打开：

```
http://localhost:5173
```

能看到 Vue 页面就可以。

然后回到总目录：

```
cd ..
```

# 第 3 步：创建 NestJS 后端

NestJS 官方推荐用 Nest CLI 创建项目。

在 `chat-app` 目录下执行：

```
npm i -g @nestjs/clinest new backend
```

它问你包管理器，选：

```
npm
```

然后进入后端项目：

```
cd backendnpm run start:dev
```

看到类似这个说明后端成功：

```
[Nest] ... LOG [NestApplication] Nest application successfully started
```

浏览器打开：

```
http://localhost:3000
```

看到：

```
Hello World!
```

就成功了。

然后回到总目录：

```
cd ..
```

# 第 4 步：添加 PostgreSQL + Redis

在 `chat-app/` 根目录创建文件：

```
docker-compose.yml
```

内容写这个：

```
services:  postgres:    image: postgres:16    container_name: chat-postgres    restart: always    environment:      POSTGRES_USER: chat_user      POSTGRES_PASSWORD: chat_password      POSTGRES_DB: chat_db    ports:      - "5432:5432"    volumes:      - postgres_data:/var/lib/postgresql/data  redis:    image: redis:7    container_name: chat-redis    restart: always    ports:      - "6379:6379"volumes:  postgres_data:
```

Postgres 官方镜像支持用环境变量设置数据库用户、密码、数据库名，其中 `POSTGRES_PASSWORD` 是必需项；Redis 也有官方 Docker 镜像。

# 第 1 步：创建总项目目录

找一个你放代码的地方，比如桌面或开发目录，然后执行：

```
mkdir chat-appcd chat-app
```

你的项目最终会长这样：

```
chat-app/
  frontend/
  backend/
  docker-compose.yml
```

# 第 2 步：创建 Vue + TS 前端

执行：

```
npm create vue@latest frontend
```

Vue 官方快速开始也是用 `npm create vue@latest` 创建项目。

它会问你一些问题，你这样选：

```
Project name: frontend
Add TypeScript? Yes
Add JSX Support? No
Add Vue Router? Yes
Add Pinia? Yes
Add Vitest? No
Add an End-to-End Testing Solution? No
Add ESLint? Yes
Add Prettier? Yes
Add Vue DevTools? Yes 或 No 都行
```

然后进入前端项目：

```
cd frontend
npm install
npm run dev
```

看到类似这个就说明前端成功：

```
Local: http://localhost:5173/
```

浏览器打开：

```
http://localhost:5173
```

能看到 Vue 页面就可以。

然后回到总目录：

```
cd ..
```

# 第 3 步：创建 NestJS 后端

NestJS 官方推荐用 Nest CLI 创建项目。

在 `chat-app` 目录下执行：

```
npm i -g @nestjs/clinest new backend
```

它问你包管理器，选：

```
npm
```

然后进入后端项目：

```
cd backendnpm run start:dev
```

看到类似这个说明后端成功：

```
[Nest] ... LOG [NestApplication] Nest application successfully started
```

浏览器打开：

```
http://localhost:3000
```

看到：

```
Hello World!
```

就成功了。

然后回到总目录：

```
cd ..
```

# 第 4 步：添加 PostgreSQL + Redis

在 `chat-app/` 根目录创建文件：

```
docker-compose.yml
```

内容写这个：

```yaml
services:
	postgres:
		image: postgres:16
		container_name: chat-postgres
		restart: always
		environment:
			POSTGRES_USER: chat_user
			POSTGRES_PASSWORD: chat_password
			POSTGRES_DB: chat_db
		ports:
			- "5432:5432"
		volumes:
			- postgres_data:/var/lib/postgresql/data
	redis:
		image: redis:7
		container_name: chat-redis
		restart: always
		ports:
			- "6379:6379"
volumes:  postgres_data:
```

Postgres 官方镜像支持用环境变量设置数据库用户、密码、数据库名，其中 `POSTGRES_PASSWORD` 是必需项；Redis 也有官方 Docker 镜像。

# 第 5 步：后端接入 Prisma

进入后端目录：

```
cd backend
```

安装 Prisma：

```
npm install prisma --save-devnpm install @prisma/client
```

初始化 Prisma：

```
npx prisma init
```

Prisma Migrate 用来管理数据库结构变更，适合后面维护表结构。

打开：

```
backend/.env
```

把数据库连接改成：

```

DATABASE_URL="postgresql://chat_user:chat_password@localhost:5432/chat_db?schema=public"
```

# 第 6 步：先建一个 User 表

打开：

```
backend/prisma/schema.prisma
```

改成这样：

```ts
generator client {
	provider = "prisma-client-js"
}

datasource db {
	provider = "postgresql"
	url = env("DATABASE_URL")
}

model User {
	id        String   @id @default(uuid())
	username  String   @unique
	password  String
	nickname  String?
	avatarUrl String?
	createdAt DateTime @default(now())
	updatedAt DateTime @updatedAt
}
```

然后执行迁移：

```
npx prisma migrate dev --name init
```

成功后，打开 Prisma Studio 看数据库：

```
npx prisma studio
```

浏览器会打开一个数据库管理页面。能看到 `User` 表，就说明后端已经连上 PostgreSQL 了。

# 第 7 步：安装后端常用依赖

先装这几个，后面做登录和聊天会用：

```
npm install @nestjs/config @nestjs/jwt passport passport-jwt bcrypt class-validator class-transformer

npm install -D @types/bcrypt @types/passport-jwt
```

然后安装 Socket.IO 相关依赖：

```
npm install @nestjs/websockets @nestjs/platform-socket.io socket.io
```

Redis 先装：

```
npm install ioredis
```

# 第 9 步：启动顺序

以后每天开发，按这个顺序启动。

第一个终端，在根目录：

```
cd chat-app
docker compose up -d
```

第二个终端，启动后端：

```
cd chat-app/backend
npm run start:dev
```

第三个终端，启动前端：

```
cd chat-app/frontend
npm run dev
```

然后打开：

```
前端：http://localhost:5173
后端：http://localhost:3000
数据库：localhost:5432
Redis：localhost:6379
```
